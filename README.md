## 1.1 Introducción a la programación de interfaces Web
La actividad consistió en el desarrollo de una aplicación web interactiva que permite visualizar y manipular colores mediante el modelo **RGB**. Para ello, se utilizaron controles deslizantes, campos numéricos y un selector de color, con los cuales el usuario puede modificar los valores de rojo, verde y azul y observar en tiempo real el color resultante.

La aplicación fue construida empleando **HTML** para la estructura, **CSS y Bootstrap** para el diseño visual, y **JavaScript** para la lógica y la sincronización de los controles. Además, se mostró el valor del color tanto en formato **RGB** como en **hexadecimal**, facilitando la comprensión de la relación entre los distintos sistemas de representación de color.
